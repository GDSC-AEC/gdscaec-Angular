30 Days of Google Cloud

 

What is this about?

 

Developer Student Club HIT in collaboration with Google Developers and Qwiklabs brings to you the #30DaysofGoogleCloud, the best place to get started with the Google Cloud Platform. Do sign up in order to have a seamless experience for #30DaysofGoogleCloud.

 

 

 

 

How do we do?

We have 2 Exciting Tracks for you. Click on a Track to get enrolled in it.

 

Cloud Engineering

Finish the following Set of Quests to complete this Track. You need to complete ALL the quests.
  • Getting Started: Create and Manage Cloud Resources
  • Perform Foundational Infrastructure Tasks in Google Cloud
  • Setup and Configure a Cloud Environment in Google Cloud
  • Deploy and Manage Your Cloud Environments with Google Cloud
  • Build and Secure Networks in Google Cloud
  • Deploy to Kubernetes in Google Cloud

 

Begin First Quest

Data Science & Machine Learning

Finish the following Set of Quests to complete this Track. You need to complete ALL the quests.
  • Getting Started: Create and Manage Cloud Resources
  • Perform Foundational Data, ML, and AI Tasks in Google Cloud
  • Insights from Data with BigQuery
  • Engineer Data in Google Cloud
  • Integrate with Machine Learning APIs
  • Explore Machine Learning Models with Explainable AI
