#30DaysofGoogleCloud

What is this about?

 

Developer Student Club AEC in collaboration with Qwiklabs brings to you the #30DaysofGoogleCloud, the best place to get started with the Google Cloud Platform. Do sign up in order to have a seamless experience for #30DaysofGoogleCloud.

 

 

How do we do?

 

Click on a Track to get enrolled in it.

 

Cloud Engineering

Finish the following Set of Quests to complete this Track. You need to complete ALL the quests.
  • Getting Started: Create and Manage Cloud Resources
  • Perform Foundational Infrastructure Tasks in Google Cloud
  • Setup and Configure a Cloud Environment in Google Cloud
  • Deploy and Manage Your Cloud Environments with Google Cloud
  • Build and Secure Networks in Google Cloud
  • Deploy to Kubernetes in Google Cloud
